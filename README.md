# easykeytopass
## propassword
Remake

### REQUIREMENTS
- Redis
- Nodejs > 8

### GETTING STARTED
Clone the project then change directory to the folder.
```bash
npm install
node main.js
```
### KNOWN ISSUES
- **md5** hashing algorithm is not the best for hashing, however, for making long password is the most suitable (32 characters) excluding the secret phrases.
